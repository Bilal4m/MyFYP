# README

## About this project?

This project is about ordering online food from your local restaurants. Anybody can create an account and order online. you are free to to use this project for college or school purposes. Please note that this project is just for learning purpose.


## Admin Dashboard

To access admin portal type this in your URL BAR **\`root**/admin/ \`  where **root** is your root directory

 **Admin credentials:** Username: **admin** Password: **1234**
